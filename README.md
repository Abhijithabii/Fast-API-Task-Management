# Task Management API

A complete task/todo management application built with FastAPI.

## Features

- ✅ Create, Read, Update, Delete tasks
- ✅ Toggle task completion status
- ✅ Filter tasks by completion status, priority, category
- ✅ Search in title and description
- ✅ Pagination support
- ✅ Task statistics
- ✅ Bulk delete completed tasks
- ✅ Category management
- ✅ SQLite database
- ✅ Auto-generated API documentation
